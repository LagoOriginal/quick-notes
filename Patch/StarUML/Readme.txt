REGISTER STAR UML v3.2.2

1. find original app.asar location, 
e.g: C:\Program Files\StarUML\resources

2. run patcher with hpatchz.exe, 
e.g: ..\hpatchz.exe -f "C:\Program Files\StarUML\resources\app.asar" star_uml_v3.2.2.patch app.asar

3. overwrite original app.asar with result file
e.g: move app.asar "C:\Program Files\StarUML\resources\app.asar"

OR
see patch.bat file at current directory
